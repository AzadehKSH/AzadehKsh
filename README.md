<h3 align="center">
  Building intelligent, efficient, and responsible AI systems for healthcare, signal processing, and real-world deployment.
</h3>

<p align="center">
  I am passionate about the intersection of <b>AI</b>, <b>cardiovascular signal processing</b>, 
  <b>edge deployment</b>, and <b>human-centered technology</b>.  
  My work focuses on developing machine learning systems that are not only accurate, 
  but also efficient, interpretable, maintainable, and useful in real clinical or embedded environments.
</p>

<p align="center">
  <b>Current focus:</b> ECG / SCG / PCG signal processing, transfer learning, pruning, lightweight models, 
  edge AI, responsible AI, and intelligent interactive systems.
</p>
